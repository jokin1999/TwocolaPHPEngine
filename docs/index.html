Twocola PHP Engine

Twocola PHP Engine简称TCE，为Web开发提供了简洁的MVC开发模式，同时拥有自主开发的窝群(OAM)系统。

  • 了解窝群(OAM)
  • 开发手册

窝群：一服多站

窝群系统简称OAM，可以帮助您在服务器只允许开设一个站点的情况下实现“一服多站”。

  • 窝群首页
  • 了解一服多站

窝群：独立Api通道

OAM可以设置独立的Api通道，同一个开发环境但访问时与应用环境隔离。

TCE 3.1

TCE 3.1版本相较于3.0版本来说是一次大规模更新，使用了全新的架构，遗憾的是不完全支持3.0版本直接升级。

独立开发

TCE框架由作者Jokin独立开发，保证了框架的运行流程被严格限制，提高了框架的运行效率。

防注入

TCE框架使用独立的数据库操作类，有防止注入攻击的方法，可直接调用进行防止注入攻击。

个性设置

TCE框架提供了全局设置和应用设置，应用设置的高优先级使您能够更专注于开发，而不是复杂的设置流程。

一服多站

TCE框架内置独创的窝群系统，能够在任何服务器上实现“一服多站”。（站点数量取决于域名可绑定数量）

Api模式

TCE内置的窝群系统可以提供单独的Api模式，本地或跨域都可以进行访问，使您免去重新开发一套系统的时间。

团队精神

TCE框架注重于团队开发的团结精神，但由于各种原因，无法建立自己的团队，但无论何时，Team First。
